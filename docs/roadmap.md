# Roadmap

## 0.1 MVP

- 静态前端演示。
- FastAPI 后端。
- Java 后端方向、难度、模式配置。
- mock 面试状态机。
- 基础报告生成。
- 前端 provider 配置保存。
- OpenAI-compatible LLM/ASR/TTS。
- Ollama LLM。
- SQLite 会话、turn、trace、报告和转写记录存储。
- YAML 题库。
- 简历文本解析。
- Silero VAD 接口。
- SenseVoice/CosyVoice 本地 runtime 适配器。
- Realtime 状态机事件接口。

## 0.2 Provider 接入

- faster-whisper ASR provider。
- Piper TTS provider。
- vLLM provider。
- provider 健康检查。
- PDF/DOCX 简历解析。

## 0.3 题库和简历

- 结构化 Java 后端题库。
- 简历项目卡片解析。
- 项目追问链：个人贡献、指标来源、技术选型、故障复盘。
- 知识点薄弱项追踪。

## 0.4 复练工作台

- 按薄弱维度生成复练题。
- 同一问题多次作答对比。
- 回答证据引用和改写建议。
- 知识点掌握度趋势。

## 0.5 多轮面试

- 一面、二面、HR 面流程。
- 不同面试官风格。
- 历史报告对比。
- 个性化复练计划。
