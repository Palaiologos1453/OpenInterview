# Question Sources

OpenInterview questions in this directory are structured interview prompts,
follow-ups, and rubrics written for the simulator. They are not verbatim copies
of the referenced material.

## JavaGuide

- Local source reviewed: `D:\JavaGuide`
- Upstream project: JavaGuide
- License: Apache License 2.0
- Usage in this directory: adapted/inspired question design with source path
  metadata on each item.

When adding new questions from JavaGuide or other materials:

1. Rewrite the prompt, follow-ups, and rubric for interview simulation.
2. Keep `source`, `source_path`, `source_license`, and `source_usage`.
3. Do not copy long answer passages into YAML.
4. Prefer scenario questions that evaluate tradeoffs, boundaries, and failure
   handling, not only memorized definitions.
